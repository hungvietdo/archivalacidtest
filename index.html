The Archival Acid Test

This page was setup to test the capabilities and shortcomings of archival web crawlers.

Why bother?

When preserving web pages, archival crawlers sometimes produce a result that varies from what an end-user expects. To quantitatively evaluate the degree to which an archival crawler is capable of comprehensively reproducing a web page from the live web into the archives, the crawlersâ€™ capabilities must be evaluated.

In this project, we propose a set of metrics to evaluate the capability of archival crawlers and other preservation tools using the Acid Test concept.
